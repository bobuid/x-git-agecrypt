*** REQ ***

Implement automatic OS-based passphrase key selection. Currently, the passphrase getter feature requires explicit specification via `-g linux` or `-g macos` argument (or env var). We want git-agecrypt to automatically detect the OS and use the appropriate key from the `[passphrase]` section when neither `-g` nor `AGE_PASSPHRASE_GETTER` is provided.

## Current Behavior

Priority order in `resolve_passphrase()`:
1. Explicit `-g <key>` argument (highest priority)
2. `AGE_PASSPHRASE_GETTER` env var
3. Implicit `sops` key fallback (lowest priority)

## Desired Behavior

Priority order:
1. Explicit `-g <key>` argument (highest priority)
2. `AGE_PASSPHRASE_GETTER` env var
3. **NEW: OS-specific key (`linux` on Linux, `macos` on macOS)**
4. Implicit `sops` key fallback (lowest priority)

## Example Config

```toml
[passphrase]
sops="sops --extract .age_passphrase secrets.enc.yaml"
linux="secret-tool lookup application age identity-file $HOME/bobs.identity"
macos="security find-generic-password -a \"age\" -s \"$HOME/bobs.identity\" -w"
```

With this config:
- On macOS: `git-agecrypt status` auto-uses the `macos` key
- On Linux: `git-agecrypt status` auto-uses the `linux` key
- Fallback to `sops` if OS-specific key not present

## Implementation Plan

1. Add `ImplicitOs` variant to `GetterSource` enum in `src/main.rs`
2. Update `Display` impl for `GetterSource` to handle `ImplicitOs`
3. Modify the fallback logic in `resolve_passphrase()` (around lines 75-83):
   - After env var check fails, determine OS-specific key name using `cfg!(target_os = ...)`
   - Check if that key exists in config
   - If yes, use it with `GetterSource::ImplicitOs`
   - If no, fall through to existing `sops` check

Err(_) => {
    // Env var not set, check OS-specific key first
    let os_key = if cfg!(target_os = "macos") { "macos" }
                 else if cfg!(target_os = "linux") { "linux" }
                 else { "" };

    if !os_key.is_empty() && cfg.has_passphrase_key(os_key) {
        (Some(os_key.to_string()), Some(GetterSource::ImplicitOs))
    } else if cfg.has_passphrase_key("sops") {
        (Some("sops".to_string()), Some(GetterSource::ImplicitSops))
    } else {
        (None, None)
    }
}
